temas : Playlist , Historico , Fila de impressao

trabalho A : Sistema para gerenciar músicas usando lista simples com vetor de struct.
Permite cadastrar, buscar, editar, excluir e listar músicas. 

Funcionalidades:
- Cadastrar música com ID automático
- Buscar por ID
- Editar mantendo o ID
- Excluir com confirmação
- Listar todas as músicas
- Salvar e carregar CSV


trabalho B: Simula os botões voltar e avançar de um navegador usando duas pilhas.
Uma pilha guarda o histórico e outra as páginas para avançar.

Funcionalidades:
- Visitar nova página (empilha no histórico)
- Voltar (move do histórico para pilha de avanço)
- Avançar (move da pilha de avanço para histórico)
- Ver página atual
- Ver histórico completo
- Data e hora reais de cada visita

trabalho C: Simula uma fila de impressão com trabalhos normais e prioritários.
Usa fila simples para normais e fila circular para prioritários para que eles sejam acessados primeiro.

Funcionalidades:
- Enviar trabalho (normal ou prioritário)
- Processar próximo (prioritário primeiro)
- Listar filas
- Buscar por ID
- Cancelar trabalho em qualquer posição
